# Round Rabbit — run and used (Fable's codex original, preserved)

`round_rabbit.py` + `mapping_web.py` are Fable's well-written codex originals,
restored verbatim from `origin/codex/v5-composition-mapping-web`. They are kept
as-is; nothing in this branch's data was clobbered.

## Running Fable's original (as written)

The typed graph is regenerated by `mapping_web.py` from `dictionary-v5.json` +
`fragments.tsv` (no MUSE needed — meaning edges come from v5's own data):

```
nodes: 15681  edges: 19120        # matches the codex census
```

Then `round_rabbit.py` builds the semantic-component → homophonic-radius lattice.
Sample (no seed, min-component-weight 0.5):

```
0 hops en:air        air -> aires; air -> airs; air -> ère
0 hops en:acoustic   acoustic -> égouts tic; acoustic -> acoustique
0 hops fr:acceptes   accept -> acceptes; except -> acceptes; accepts -> acceptes
```

It works well: from a meaning it returns the bilingual homophone strings in its
sound radius — a themed homophone field. (Run it in a scratch dir so its
`mapping-web.json` output does not overwrite this branch's differently-formatted
`mapping-web.json`.)

## Branch-native runner (this branch's graph)

`round_rabbit_run.py` runs the same Round Rabbit idea directly on **this**
branch's `mapping-web.json` (sound + meaning adjacency), with the **best-product
walk** (Dijkstra in the product semiring — the Fix 1 validated in
`DEPS_RABBIT_AND_NOVEL.md`, choosing the highest-confidence path, not fewest
hops). Results are genuinely good:

| theme | homophone field (sample) |
|---|---|
| king queen | queen → couine / couinent / cousine |
| water hill | clean/calling/cleaner → colline; clean → incline |
| moon star | star → stars; starter → stars |

entries included.

## How it fits the generation engine

Round Rabbit is the **content-selection front end** (impetus III): pick a meaning,
get the bilingual sound-neighbours that are *already homophonic*, and feed those
as the high-yield material for the whole-line carve. It answers "what should I say
here so the languages are in tune" — exactly the lever the failing nursery lines
(Jack and Jill, Hickory) needed. Use Fable's lattice to choose carve-able content,
then the carve engine to render it.
