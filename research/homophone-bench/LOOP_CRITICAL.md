# Loop-certified pairs: Fable-critical lens

Claim (prior): a loop certifies meaning survived the round trip, so every sound
edge in a loop = meaning-preserving homophone pair. 121 S-tier loop-certified.

CRITICAL TEST: does the loop contain a real = (translation) hop, or does it close
via ~ (synonym) drift?
  MEANING-GROUNDED (has =): 19 / 121   <- real dual-translation gold
  sound+synonym only:       102 / 121  <- homophone on a loop, NOT a translation

So the claim is OVERCLAIMED. The loop certifies TOPOLOGICAL return, not pair
meaning. 102/121 close through synonym hops (meaning drifts) -- e.g. succeed~secs
ides, clack~claque: perfect homophones, zero meaning relation. Only the 19 with a
= anchor (country=pays, kindly=gentiment, poop=crotte) are genuine sound+sense
pairs.

## What's actually gold

- loop-grounded-gold.tsv (19): = -anchored = real homophonic+semantic translation
  pairs. THIS is the no-LLM dual-translation seed, not the full 121.
- the other 102: trusted CHAIN edges (good for routing/composition), but sound-
  only -- use for homophonic wordplay, not semantic translation.

## Bootstrap, corrected

certify -> promote -> re-weave compounds CONNECTIVITY (more loops), but semantic
gold grows only with the = -anchored subset, which is small (19) and bounded by
the translation-edge density (MUSE/embeddings). So the bootstrap mines trusted
sound topology for free; the SEMANTIC gold still needs the meaning layer grown.
Same gap, now quantified: 16% of loop-certs are meaning-grounded.
